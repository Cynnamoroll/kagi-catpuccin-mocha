# Kagi Catpuccin Mocha

A soothing pastel theme for Kagi search and assistant. Based on the official Catpuccin Mocha colour scheme.

Simply copy the CSS file contents into the [Kagi appearance custom CSS page](https://kagi.com/settings/custom_css).
